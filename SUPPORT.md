# Support

For bugs, feature requests, and usage questions, please open an issue:

<https://github.com/occasional16/quicklook-for-vscode/issues>

Before opening an issue, please include:

- VS Code version.
- Extension version.
- Windows version.
- QuickLook installation path.
- Output from the `QuickLook` output channel, if preview fails.
- Whether QuickLook can preview the same file outside VS Code.

Do not paste secrets, tokens, private file contents, or sensitive paths into public issues.
