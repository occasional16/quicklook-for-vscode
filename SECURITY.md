# Security Policy

## Supported Versions

The latest Marketplace release is supported.

## Reporting a Vulnerability

Please report security issues privately instead of opening a public issue.

If GitHub private vulnerability reporting is enabled for this repository, use that channel first. Otherwise, contact the repository owner through GitHub and provide a minimal description of the issue without sharing exploit details publicly.

Do not include secrets, tokens, or private file contents in public reports.

## Scope

This extension launches the local QuickLook executable with a selected local file path. It does not upload files, collect telemetry, or send file paths to external services.
